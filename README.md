# AI 股票穿透分析系统

## 项目简介

AI 股票穿透分析系统是一款基于 DeepSeek AI 的智能股票分析工具，集成了实时市场数据、资金流向分析、技术指标计算、新闻资讯获取等功能，为投资者提供全面、专业的股票分析报告。

## 核心功能

### 1. 股票/基金分析
- 实时盘口数据获取（基于腾讯财经接口）
- 资金流向分析（超大单、大单、中单、小单）
- 技术指标计算（EMA均线、RSI、KDJ、布林带）
- 板块信息分析
- 上证指数对比分析

### 2. AI 智能分析
- 基于 DeepSeek AI 的深度分析
- 自动生成专业分析报告
- 投资建议和风险控制策略
- 支持标准模式和思考模式两种分析模式

### 3. 资讯获取与分析
- 新浪财经最新资讯获取
- 资讯详情智能解析
- 支持获取全部资讯正文内容
- 资讯与股票相关性分析

### 4. 用户系统
- 用户注册与登录
- 积分管理系统
- 历史分析记录管理
- 个人信息修改

### 5. 后台管理
- 系统设置管理
- 用户管理
- 分析记录管理
- 系统日志查看

### 6. 其他功能
- 响应式设计，支持移动端
- 深色/浅色模式切换
- 分析报告导出（PDF、图片）
- 复盘数据录入与分析

## 技术栈

### 前端
- HTML5 + CSS3
- JavaScript
- Tailwind CSS
- Chart.js
- jsPDF + html2canvas

### 后端
- PHP 7.4+
- MySQL
- cURL（数据获取）
- DeepSeek API（AI 分析）

### 数据来源
- 腾讯财经（实时盘口数据）
- 新浪财经（最新资讯）
- 东方财富（板块信息、资金流向）

## 安装部署

### 环境要求
- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本
- Web 服务器（Apache/Nginx）
- PHP 扩展：curl, mbstring, json

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/robotcn1130/gp.git
   cd gp
   ```

2. **配置数据库**
   - 创建 MySQL 数据库
   - 导入数据库结构（可通过 install.php 自动创建）

3. **配置系统参数**
   - 编辑 `config.php` 文件，设置数据库连接信息
   - 设置 DeepSeek API Key

4. **运行安装脚本**
   ```bash
   # 在浏览器中访问
   http://your-domain/gp/install.php
   ```

5. **完成安装**
   - 创建管理员账户
   - 配置系统基本参数

## 使用说明

### 1. 股票分析流程

1. **登录系统**
   - 点击右上角的"登录"按钮
   - 输入用户名和密码

2. **输入股票代码**
   - 在搜索框中输入股票名称或代码
   - 选择对应的股票

3. **设置分析参数**
   - 输入持有数量和成本（可选）
   - 选择分析模型（标准模式/思考模式）
   - 选择是否获取全部资讯内容

4. **开始分析**
   - 点击"开始实时数据分析"按钮
   - 系统将自动获取数据并生成分析报告

5. **查看分析结果**
   - 查看实时盘口数据
   - 查看资金流向分析
   - 查看 AI 生成的分析报告
   - 导出分析报告（可选）

### 2. 后台管理

1. **访问后台**
   ```bash
   http://your-domain/gp/admin.html
   ```

2. **登录后台**
   - 使用管理员账户登录

3. **管理系统**
   - 管理用户账户
   - 查看系统日志
   - 配置系统参数

## 系统设置

### 1. DeepSeek API Key 配置
- 在后台管理页面的系统设置中配置
- 或直接编辑 `config.php` 文件

### 2. 分析成本设置
- 在后台管理页面设置不同模型的分析积分成本
- 标准模式：默认 1 积分/次
- 思考模式：默认 2 积分/次

### 3. 充值设置
- 在后台管理页面配置充值二维码和说明

## 注意事项

1. **数据来源**
   - 本系统数据来源于公开渠道，仅供参考
   - 数据更新可能存在延迟

2. **AI 分析**
   - AI 分析结果仅供参考，不构成投资建议
   - 投资决策请结合自身情况综合考虑

3. **积分系统**
   - 分析功能需要消耗积分
   - 新用户注册后可获得初始积分
   - 可通过充值获取更多积分

4. **系统安全**
   - 请勿泄露个人账号信息
   - 定期修改密码
   - 确保服务器环境安全
